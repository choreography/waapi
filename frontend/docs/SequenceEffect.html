SequenceEffect

Provides a way to group multiple animation effects together, similar to GroupEffect.

When animated it synchronises effects to play one after the other has finished, as per the order defined from the effects parameter.

var group = new SequenceEffect(effects); var effects = [KeyframeEffect, GroupEffect, SequenceEffect /*, ... */];

Arguments

  1. effects

    are a group of animations played in order.
/// Constructing and playing a SequenceEffect made of two effects var element1 = document.querySelector('.animate1'); var element2 = document.querySelector('.animate2'); var keyframes = [ { opacity: 1 }, { opacity: 0 } ]; // Fade Out var timings = { duration: 500 }; var effects = [ new KeyframeEffect(element1, keyframes, timings), new KeyframeEffect(element2, keyframes, timings) ]; var group = new SequenceEffect(effects); document.timeline.play(group);
