GroupEffect

Provides a way to group multiple animation effects together, similar to SequenceEffect.

When animated it synchronises to start all the effects all at the same time.

var group = new GroupEffect(effects); var effects = [KeyframeEffect, GroupEffect, SequenceEffect /*, ... */];

Arguments

  1. effects

    are a group of animations played together at once.
/// Constructing and playing a GroupEffect made of two effects var element1 = document.querySelector('.animate1'); var element2 = document.querySelector('.animate2'); var keyframes = [ { opacity: 1 }, { opacity: 0 } ]; var timings = { duration: 500 }; var effects = [ new KeyframeEffect(element1, keyframes, timings), new KeyframeEffect(element2, keyframes, timings) ]; var group = new GroupEffect(effects); document.timeline.play(group);
