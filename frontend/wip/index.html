Web Animations API

KeyframeEffect

KeyframeEffect updates CSS properties of a target element over time.

They sit within an Animation and work with the global clock known as the DocumentTimeline.

new

KeyframeEffect

(target, frames, options);

Arguments

  1. target

    is the Element being animated.
    Can be null.

  2. frames

    is an array of Keyframes to use.

  3. options

    is either a Number specifying duration in milliseconds OR an Object with properties specifying timing and behaviour of the effect.

    For a list & description of the properties see KeyframeEffectOptions.

The target can be null for animations that don't target a specific element, for example an animation that makes sound using the audio API.

/// Constructing a KeyframeEffect // Grab a DOM element to animate var target = document.querySelector('.my-target'); // Keyframes for fade out var frames = [ { opacity: 1 }, { opacity: 0 } ]; // Play effect for 300ms, delay it for 100ms, keep CSS active beyond animation var options = { delay: 100, duration: 300, fill: 'both' }; var effect = new KeyframeEffect(target, frames, options); /// You can use the effect by putting it into an animation var animation = document.timeline.play(effect); /// Alternative, you can composite it with other effects var group = new GroupEffect([ effect, otherEffect ]); var animation = document.timeline.play(group);

Examples

Example var fadeOutEffect = new KeyframeEffect(document.querySelector('div.test'), [
{ opacity: 1 },
{ opacity: 0 }
], 500);
