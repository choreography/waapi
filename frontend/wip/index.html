KeyframeEffect

(target, frames, options);

KeyframeEffects update CSS properties of a target element over time.

They sit within an Animation and work with the global clock known as the DocumentTimeline

  1. target

    = Element

    The target element being animated. Can be null for animations that don't target a specific element, for example an animation that makes sound using the audio API.

  2. frames

    = Array<Keyframes>

    An array of keyframes to use.

  3. options

    = Number|Object<KeyframeEffectOptions>

    Either a number specifying duration in milliseconds OR an object with properties specifying timing and behaviour of the effect.
